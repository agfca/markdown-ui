Markdown UI

Responsive UI in Markdown

Installation

gem install markdown-ui

Usage

markdown-ui index.mdui > index.html

Credits

Markdown-UI uses Semantic-UI http://www.semantic-ui.com syntax and framework, and ruby redcarpet library. Credit goes to the people behind these wonderful framework and libraries.

Issues/Bugs

  • Ongoing support for 80% of Semantic-UI elements/modules/components
  • The Colon (:) character will be parsed when used inside a text, needs post-processing to dislay correctly (for URLs)
  • Code is not DRY, needs refactoring / rewriting
  • Implimentation is subject to change
